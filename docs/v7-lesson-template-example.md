## 🎬 ABA: ROTEIRO

### Roteiro Narrativo Completo
```
Você já se perguntou qual IA usar para cada tarefa?

Claude ou ChatGPT? Gemini? Perplexity?

Cada modelo tem seus superpoderes. E escolher o certo pode fazer TODA a diferença no resultado.

Hoje vamos focar em Claude e ChatGPT. Os dois mais populares.

E vou te mostrar EXATAMENTE quando usar cada um.


[ATO 1: APRESENTAÇÃO DOS MODELOS]

Claude é o analista profundo. Ele brilha em:
- Análise de documentos longos (até 200 mil palavras!)
- Raciocínio complexo passo a passo
- Precisão em tarefas técnicas
- Seguir instruções detalhadas à risca

ChatGPT é o criativo versátil. Ele domina em:
- Geração criativa de conteúdo
- Conversas naturais e fluidas
- Brainstorming e ideação
- Tarefas gerais diversas


[ATO 2: COMPARAÇÃO PRÁTICA]

Vou te mostrar 3 cenários REAIS onde a escolha importa:

CENÁRIO 1: Analisar um contrato de 50 páginas
→ Claude: Lê TUDO, identifica cláusulas problemáticas, explica em detalhes
→ ChatGPT: Pode perder contexto em documentos muito longos, resumo mais superficial

Vencedor: CLAUDE ✓

CENÁRIO 2: Escrever copy criativo para Instagram
→ Claude: Correto e bem escrito, mas mais formal
→ ChatGPT: Criativo, natural, engajante, linguagem jovem

Vencedor: CHATGPT ✓

CENÁRIO 3: Resolver um problema de programação complexo
→ Claude: Raciocínio passo a passo, explicações detalhadas de cada linha
→ ChatGPT: Solução rápida e funcional, mas pode pular explicações

Vencedor: CLAUDE ✓


[ATO 3: DESAFIO INTERATIVO]

Agora é sua vez de testar seu conhecimento!

Vou te dar 5 tarefas do mundo real.
Para cada uma, você escolhe: Claude ou ChatGPT?

Pense bem antes de responder!

1. Revisar uma tese de mestrado de 200 páginas
2. Criar 10 ideias criativas de posts para TikTok
3. Analisar planilha de dados financeiros e encontrar padrões
4. Escrever um roteiro de podcast divertido e descontraído
5. Fazer debug de código Python complexo com múltiplos erros

Fez suas escolhas? Vamos ver as respostas!


[ATO 4: RESPOSTA E APRENDIZADO]

Aqui estão as melhores escolhas e o PORQUÊ:

1. Revisar tese de mestrado → CLAUDE ✓
   Por quê? Contexto longo (200 páginas) + análise profunda + precisão acadêmica

2. Posts para TikTok → CHATGPT ✓
   Por quê? Criatividade + linguagem jovem + geração rápida de ideias

3. Análise de dados financeiros → CLAUDE ✓
   Por quê? Análise rigorosa + identificação de padrões + precisão numérica

4. Roteiro de podcast → CHATGPT ✓
   Por quê? Tom conversacional + criatividade + fluidez natural

5. Debug de Python → CLAUDE ✓
   Por quê? Raciocínio técnico detalhado + explicação passo a passo + precisão


Quantas você acertou?
- 5/5: Você domina! 🏆
- 3-4: Muito bem! Está no caminho certo! ✨
- 1-2: Sem problemas! Agora você aprendeu! 💪


[ATO 5: REGRA DE OURO]

A regra é simples e você nunca mais vai esquecer:

📊 Claude = Profundidade, Análise, Precisão
→ Use para: documentos longos, análise técnica, raciocínio complexo

✨ ChatGPT = Criatividade, Versatilidade, Conversação
→ Use para: conteúdo criativo, brainstorming, tarefas gerais

E sabe o melhor?
Você NÃO precisa escolher apenas um!

Use os DOIS juntos:
1. Claude para analisar e estruturar
2. ChatGPT para criar e polir

Exemplo prático:
- Claude analisa dados e cria estrutura de artigo
- ChatGPT pega a estrutura e escreve com criatividade

É a combinação PERFEITA! 🎯


[ENCERRAMENTO]

Agora você tem o poder da escolha certa.

Não existe IA "melhor".
Existe a IA CERTA para cada tarefa.

Você acabou de aprender:
✓ Quando usar Claude
✓ Quando usar ChatGPT
✓ Como combinar os dois
✓ A regra de ouro para nunca mais errar

Na próxima aula: "Gemini vs Perplexity - O Confronto dos Pesquisadores"

Você vai descobrir qual IA domina em pesquisa e busca de informações!

Até lá! 🚀
```

## 💡 DICAS PARA CRIAR ROTEIROS V7

1. **Estruture em atos claros**: Introdução → Conceito → Prática → Aprendizado → Encerramento
2. **Use comparações**: "X vs Y", "Antes e Depois", "Amador vs Profissional"
3. **Inclua elementos interativos**: Quizzes, desafios, escolhas
4. **Seja conversacional**: Escreva como se estivesse falando diretamente com o aluno
5. **Crie ganchos**: Perguntas que criam curiosidade
6. **Dê exemplos práticos**: Cenários do mundo real
7. **Encerre com clareza**: Resumo do que foi aprendido + próximos passos
